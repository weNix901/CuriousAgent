## Architecture Overview

```
Python Process (curious_agent.py)
├── SpiderAgent (Thread)
│   ├── Continuous exploration
│   ├── Consumes DreamInbox
│   └── Notifies DreamAgent via queue
├── DreamAgent (Thread)
│   ├── High-priority queue (5s timeout)
│   ├── Low-priority round-robin
│   └── Creative insight generation
├── SleepPruner (Thread)
│   ├── Adaptive interval (4h → 24h)
│   └── 5 dormancy criteria
└── Main Thread
    └── Status monitoring

Communication:
- queue.Queue: SpiderAgent → DreamAgent
- SharedInbox (JSON): DreamAgent → SpiderAgent
- NodeLockRegistry: Thread-safe KG access
```

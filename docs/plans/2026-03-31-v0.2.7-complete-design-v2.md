### 2.2 核心设计决策修订

#### 决策 1: 分段锁替代全局锁

**原方案**: StateMachine 使用全局锁  
**问题**: 性能瓶颈  
**修订**: 使用分段锁（按 topic hash 分段）

```python
class SegmentedLock:
    """分段锁 - 减少锁竞争"""
    
    def __init__(self, segment_count: int = 16):
        self._segments = [threading.Lock() for _ in range(segment_count)]
        self._segment_count = segment_count
    
    def _get_segment(self, key: str) -> int:
        return hash(key) % self._segment_count
    
    def acquire(self, key: str):
        return self._segments[self._get_segment(key)].acquire()
    
    def release(self, key: str):
        return self._segments[self._get_segment(key)].release()
    
    def __enter__(self, key: str):
        self.acquire(key)
        return self
    
    def __exit__(self, *args):
        # 需要知道是哪个 key，实际使用稍复杂
        pass
```

#### 决策 2: 文件级并发控制

**新增 FileLockManager**:

```python
import fcntl  # Unix文件锁
import portalocker  # 跨平台文件锁

class FileLockManager:
    """
    进程级文件锁管理器
    
    解决多进程同时读写 JSON 文件的问题
    """
    
    def __init__(self, lock_file: str):
        self._lock_file = lock_file
        self._fd = None
    
    def acquire_read_lock(self):
        """获取读锁（共享锁）"""
        self._fd = open(self._lock_file, 'w')
        portalocker.lock(self._fd, portalocker.LOCK_SH)
    
    def acquire_write_lock(self):
        """获取写锁（排他锁）"""
        self._fd = open(self._lock_file, 'w')
        portalocker.lock(self._fd, portalocker.LOCK_EX)
    
    def release(self):
        """释放锁"""
        if self._fd:
            portalocker.unlock(self._fd)
            self._fd.close()
            self._fd = None
    
    def __enter__(self, lock_type='read'):
        if lock_type == 'read':
            self.acquire_read_lock()
        else:
            self.acquire_write_lock()
        return self
    
    def __exit__(self, *args):
        self.release()
```

class StateRepository:
    """
    状态仓库 - 负责状态数据的持久化
    
    使用 JSON 文件存储，支持：
    - 原子写操作
    - 自动备份
    - 版本控制
    """
    
    VERSION = "2.0"
    
    def __init__(
        self,
        data_dir: str,
        file_lock_manager: FileLockManager,
        backup_manager: 'BackupManager',
    ):
        self._data_dir = data_dir
        self._state_file = os.path.join(data_dir, "state_v2.json")
        self._lock_mgr = file_lock_manager
        self._backup_mgr = backup_manager
        self._cache = {}  # 内存缓存
        self._cache_lock = threading.RLock()
        
        self._init_storage()
    
    def _init_storage(self):
        """初始化存储"""
        if not os.path.exists(self._state_file):
            self._write_initial_state()
    
    def _write_initial_state(self):
        """写入初始状态"""
        initial = {
            "version": self.VERSION,
            "created_at": datetime.now().isoformat(),
            "topics": {},  # topic -> StateInfo
            "transitions": [],  # 全局状态历史（可选）
            "metadata": {},
        }
        self._atomic_write(initial)
    
    def _atomic_write(self, data: dict):
        """原子写入 - 防止写损坏"""
        temp_file = f"{self._state_file}.tmp"
        
        with self._lock_mgr(lock_type='write'):
            # 先备份
            if os.path.exists(self._state_file):
                self._backup_mgr.create_backup(self._state_file)
            
            # 写入临时文件
            with open(temp_file, 'w', encoding='utf-8') as f:
                json.dump(data, f, ensure_ascii=False, indent=2)
                f.flush()
                os.fsync(f.fileno())  # 确保写入磁盘
            
            # 原子重命名
            os.rename(temp_file, self._state_file)
    
    def _read_state(self) -> dict:
        """读取状态"""
        with self._lock_mgr(lock_type='read'):
            with open(self._state_file, 'r', encoding='utf-8') as f:
                return json.load(f)
    
    def get_state(self, topic: str) -> Optional[StateInfo]:
        """获取话题状态"""
        # 先查缓存
        with self._cache_lock:
            if topic in self._cache:
                return self._cache[topic]
        
        # 查文件
        data = self._read_state()
        topic_data = data.get("topics", {}).get(topic)
        
        if topic_data:
            state_info = StateInfo.from_dict(topic_data)
            # 更新缓存
            with self._cache_lock:
                self._cache[topic] = state_info
            return state_info
        
        return None
    
    def save_state(self, state_info: StateInfo):
        """保存状态"""
        # 更新缓存
        with self._cache_lock:
            self._cache[state_info.topic] = state_info
        
        # 更新文件
        data = self._read_state()
        data["topics"][state_info.topic] = state_info.to_dict()
        self._atomic_write(data)
    
    def get_all_states(self) -> Dict[str, StateInfo]:
        """获取所有状态"""
        data = self._read_state()
        return {
            topic: StateInfo.from_dict(info)
            for topic, info in data.get("topics", {}).items()
        }
    
    def clear_cache(self):
        """清空缓存"""
        with self._cache_lock:
            self._cache.clear()
```

class QueueRepository:
    """
    队列仓库 - 负责队列数据的持久化
    """
    
    VERSION = "2.0"
    
    def __init__(
        self,
        data_dir: str,
        file_lock_manager: FileLockManager,
        backup_manager: 'BackupManager',
    ):
        self._data_dir = data_dir
        self._queue_file = os.path.join(data_dir, "curiosity_queue_v2.json")
        self._lock_mgr = file_lock_manager
        self._backup_mgr = backup_manager
        
        self._init_storage()
    
    def _init_storage(self):
        """初始化存储"""
        if not os.path.exists(self._queue_file):
            self._write_initial_queue()
    
    def _write_initial_queue(self):
        """写入初始队列"""
        initial = {
            "version": self.VERSION,
            "created_at": datetime.now().isoformat(),
            "items": [],
            "sequence": 0,  # 自增序列号
            "metadata": {},
        }
        self._atomic_write(initial)
    
    def _atomic_write(self, data: dict):
        """原子写入"""
        temp_file = f"{self._queue_file}.tmp"
        
        with self._lock_mgr(lock_type='write'):
            if os.path.exists(self._queue_file):
                self._backup_mgr.create_backup(self._queue_file)
            
            with open(temp_file, 'w', encoding='utf-8') as f:
                json.dump(data, f, ensure_ascii=False, indent=2)
                f.flush()
                os.fsync(f.fileno())
            
            os.rename(temp_file, self._queue_file)
    
    def _read_queue(self) -> dict:
        """读取队列"""
        with self._lock_mgr(lock_type='read'):
            with open(self._queue_file, 'r', encoding='utf-8') as f:
                return json.load(f)
    
    def save(self, item: QueueItem):
        """保存队列项"""
        data = self._read_queue()
        
        # 查找并更新，或新增
        items = data.get("items", [])
        for i, existing in enumerate(items):
            if existing["id"] == item.id:
                items[i] = item.to_dict()
                break
        else:
            items.append(item.to_dict())
        
        data["items"] = items
        self._atomic_write(data)
    
    def get_by_id(self, item_id: str) -> Optional[QueueItem]:
        """通过ID获取"""
        data = self._read_queue()
        for item_data in data.get("items", []):
            if item_data["id"] == item_id:
                return QueueItem.from_dict(item_data)
        return None
    
    def find_by_status(self, status: State) -> List[QueueItem]:
        """按状态查找"""
        data = self._read_queue()
        return [
            QueueItem.from_dict(item_data)
            for item_data in data.get("items", [])
            if item_data.get("status") == status.value
        ]
    
    def count_all(self) -> int:
        """总数"""
        data = self._read_queue()
        return len(data.get("items", []))
    
    def get_next_sequence(self) -> int:
        """获取下一个序列号（原子操作）"""
        with self._lock_mgr(lock_type='write'):
            data = self._read_queue()
            seq = data.get("sequence", 0) + 1
            data["sequence"] = seq
            self._atomic_write(data)
            return seq
```

### 3.2 超时处理机制（新增）

```python
class TimeoutMonitor:
    """
    超时监控器
    
    定期检查各状态是否超时，触发相应处理
    """
    
    CHECK_INTERVAL = 30  # 秒
    
    def __init__(
        self,
        state_machine: ExplorationStateMachine,
        queue_service: QueueService,
    ):
        self._sm = state_machine
        self._queue = queue_service
        self._running = False
        self._thread = None
        self._handlers: Dict[State, Callable] = {}
        
        self._register_default_handlers()
    
    def _register_default_handlers(self):
        """注册默认超时处理器"""
        self._handlers[State.CLAIMED] = self._handle_claimed_timeout
        self._handlers[State.EXPLORING] = self._handle_exploring_timeout
    
    def start(self):
        """启动监控"""
        self._running = True
        self._thread = threading.Thread(target=self._monitor_loop, daemon=True)
        self._thread.start()
        logger.info("[TimeoutMonitor] Started")
    
    def stop(self):
        """停止监控"""
        self._running = False
        if self._thread:
            self._thread.join(timeout=5)
        logger.info("[TimeoutMonitor] Stopped")
    
    def _monitor_loop(self):
        """监控循环"""
        while self._running:
            try:
                self._check_timeouts()
            except Exception as e:
                logger.error(f"[TimeoutMonitor] Error: {e}")
            
            time.sleep(self.CHECK_INTERVAL)
    
    def _check_timeouts(self):
        """检查所有超时"""
        now = datetime.now()
        
        # 获取所有非终态话题
        for state in [State.CLAIMED, State.EXPLORING]:
            items = self._queue.find_by_status(state)
            
            for item in items:
                if self._is_timeout(item, now):
                    handler = self._handlers.get(state)
                    if handler:
                        handler(item)
    
    def _is_timeout(self, item: QueueItem, now: datetime) -> bool:
        """检查是否超时"""
        # 获取进入当前状态的时间
        current_state_time = None
        for transition in reversed(item.state_history):
            if transition.to_state == item.status:
                current_state_time = transition.timestamp
                break
        
        if not current_state_time:
            return False
        
        elapsed = (now - current_state_time).total_seconds()
        
        # 根据状态获取超时时间
        config = ExplorationStateMachine.STATE_CONFIG.get(item.status)
        if config and config.timeout_seconds:
            return elapsed > config.timeout_seconds
        
        return False
    
    def _handle_claimed_timeout(self, item: QueueItem):
        """处理 CLAIMED 超时 - 释放回 PENDING"""
        logger.warning(f"[TimeoutMonitor] CLAIMED timeout for {item.topic}")
        
        # 状态转换：CLAIMED -> PENDING
        result = self._sm.transition(
            topic=item.topic,
            to_state=State.PENDING,
            actor=Actor.SYSTEM,
            reason="CLAIMED timeout - auto release",
        )
        
        if result.success:
            item.status = State.PENDING
            item.metadata["timeout_released_at"] = datetime.now().isoformat()
            self._queue._repo.save(item)
    
    def _handle_exploring_timeout(self, item: QueueItem):
        """处理 EXPLORING 超时 - 标记为 FAILED"""
        logger.warning(f"[TimeoutMonitor] EXPLORING timeout for {item.topic}")
        
        # 状态转换：EXPLORING -> FAILED
        result = self._sm.transition(
            topic=item.topic,
            to_state=State.FAILED,
            actor=Actor.SYSTEM,
            reason="EXPLORING timeout - marked as failed",
        )
        
        if result.success:
            item.status = State.FAILED
            item.metadata["timeout_failed_at"] = datetime.now().isoformat()
            self._queue._repo.save(item)
```

class DataMigration:

    """
    
    def __init__(
        self,

    def migrate(self, dry_run: bool = False) -> 'MigrationResult':
        """
        执行迁移
        
        Args:
            dry_run: 如果为True，只模拟不实际写入
            
        Returns:
            MigrationResult: 迁移结果统计
        """
        result = MigrationResult()
        
        # 1. 备份原始数据
        if not dry_run:
            self._backup_original_data()
        
        # 2. 迁移队列数据
        queue_items = self._kg_repo.get_curiosity_queue()
        
        for old_item in queue_items:
            try:
                new_item = self._convert_queue_item(old_item)
                
                if not dry_run:
                    self._queue_repo.save(new_item)
                    
                    # 同时创建状态记录
                    state_info = self._create_initial_state(new_item)
                    self._state_repo.save_state(state_info)
                
                result.migrated += 1
                
            except Exception as e:
                result.failed += 1
                result.errors.append(f"Failed to migrate {old_item}: {e}")
        
        # 3. 验证迁移结果
        if not dry_run:
            result.validation = self._validate_migration()
        
        return result
    
    def _convert_queue_item(self, old_item: dict) -> QueueItem:
        """转换单个队列项"""
        # 从旧格式提取信息
        topic = old_item.get("topic")
        status_str = old_item.get("status", "pending")
        score = old_item.get("relevance", 0.0)
        depth = old_item.get("depth", 5.0)
        
        # 尝试推断 parent（从KG中查）
        parent_topic = None
        kg_topic = self._kg_repo.get_topic(topic)
        if kg_topic:
            parents = kg_topic.get("parents", [])
            if parents:
                parent_topic = parents[0]  # 取第一个parent
        
        lineage = LineageInfo(
            parent_topic=parent_topic,
            injected_by=old_item.get("source", "unknown"),
            injected_at=datetime.fromisoformat(old_item.get("created_at")) 
                if "created_at" in old_item else datetime.now(),
            original_reason=old_item.get("reason", ""),
        )
        
        return QueueItem(
            topic=topic,
            status=State(status_str),
            lineage=lineage,
            score=score,
            depth=depth,
            created_at=datetime.fromisoformat(old_item.get("created_at")) 
                if "created_at" in old_item else datetime.now(),
        )
    
    def _create_initial_state(self, item: QueueItem) -> StateInfo:
        """为迁移的项创建初始状态"""
        return StateInfo(
            topic=item.topic,
            state=item.status,
            history=[
                StateTransition(
                    from_state=None,
                    to_state=item.status,
                    actor=Actor.SYSTEM,
                    reason="Migrated from v1",
                )
            ],
        )
    
    def _backup_original_data(self):
        """备份原始数据"""
        backup_dir = f"knowledge/backup/migration_{datetime.now().strftime('%Y%m%d_%H%M%S')}"
        os.makedirs(backup_dir, exist_ok=True)
        
        # 复制原文件
        import shutil
        shutil.copy("knowledge/state.json", backup_dir)
        
        logger.info(f"[Migration] Original data backed up to {backup_dir}")
    
    def _validate_migration(self) -> 'ValidationResult':
        """验证迁移结果"""
        # 检查数量是否一致
        old_count = len(self._kg_repo.get_curiosity_queue())
        new_count = self._queue_repo.count_all()
        
        return ValidationResult(
            old_count=old_count,
            new_count=new_count,
            consistent=(old_count == new_count),
        )

    """

    清理旧的状态历史，防止内存无限增长
    """
    
    def __init__(
        self,
        state_repository: StateRepository,
        max_history_per_topic: int = 100,
        archive_after_days: int = 30,
    ):
        self._repo = state_repository
        self._max_history = max_history_per_topic
        self._archive_days = archive_after_days
    
    def archive_old_states(self):
        """归档旧的状态"""
        cutoff = datetime.now() - timedelta(days=self._archive_days)
        
        all_states = self._repo.get_all_states()
        
        for topic, state_info in all_states.items():
            # 只保留最近 N 条历史
            if len(state_info.history) > self._max_history:
                # 归档旧的历史
                to_archive = state_info.history[:-self._max_history]
                self._archive_history(topic, to_archive)
                
                # 保留最近的历史
                state_info.history = state_info.history[-self._max_history:]
                self._repo.save_state(state_info)
    
    def _archive_history(self, topic: str, history: List[StateTransition]):
        """将历史归档到单独文件"""
        archive_file = f"knowledge/archive/{topic}_{datetime.now().strftime('%Y%m')}.json"
        os.makedirs(os.path.dirname(archive_file), exist_ok=True)
        
        archive_data = {
            "topic": topic,
            "archived_at": datetime.now().isoformat(),
            "transitions": [t.to_dict() for t in history],
        }
        
        with open(archive_file, 'a') as f:
            f.write(json.dumps(archive_data) + "\n")

# core/quality_v2.py

class QualityV2Assessor:
    """
    质量评估器 v2 - 修复 G4
    
    使用懒加载模式，自动获取 KG 实例
    """
    
    def __init__(self, llm_client: 'LLMClient'):
        self._llm = llm_client
        self._kg_module = None
        self._lock = threading.Lock()
    
    @property
    def kg(self):
        """懒加载 KG 模块"""
        if self._kg_module is None:
            with self._lock:
                # 双重检查锁定
                if self._kg_module is None:
                    from core import knowledge_graph as kg_module
                    self._kg_module = kg_module
        return self._kg_module
    
    def assess_quality(
        self,
        topic: str,
        findings: dict,
        knowledge_graph=None,  # 向后兼容，但不再必需
    ) -> float:
        """
        评估探索质量
        
        Args:
            topic: 话题
            findings: 探索结果
            knowledge_graph: 可选，如果不传则使用懒加载的实例
            
        Returns:
            float: 质量分数 0-10
        """
        try:
            # 优先使用传入的，否则用懒加载的
            kg = knowledge_graph or self.kg
            
            if kg is None:
                logger.error("[QualityV2] Knowledge graph not available")
                return 0.0
            
            # 获取之前的内容摘要
            prev_summary = self._get_previous_summary(topic, kg)
            
            # 计算语义新颖度
            novelty = self._calculate_novelty(prev_summary, findings.get("summary", ""))
            
            # 计算信息增益
            info_gain = self._calculate_info_gain(topic, findings, kg)
            
            # 综合评分
            quality = self._compute_quality_score(novelty, info_gain)
            
            logger.debug(f"[QualityV2] {topic}: novelty={novelty:.2f}, gain={info_gain:.2f}, quality={quality:.2f}")
            
            return quality
            
        except Exception as e:
            logger.error(f"[QualityV2] Error assessing {topic}: {e}")
            return 0.0
    
    def _get_previous_summary(self, topic: str, kg) -> str:
        """获取之前的内容摘要"""
        try:
            # 先从KG获取
            topic_data = kg.get_topic(topic)
            if topic_data:
                return topic_data.get("summary", "")
            
            # 再从探索历史获取
            history = kg.get_exploration_history(topic)
            if history:
                return history[-1].get("summary", "")
            
            return ""
            
        except Exception as e:
            logger.warning(f"[QualityV2] Error getting previous summary: {e}")
            return ""
    
    def _calculate_novelty(self, prev_summary: str, new_summary: str) -> float:
        """计算语义新颖度"""
        if not prev_summary:
            return 1.0  # 全新内容
        
        if not new_summary:
            return 0.0  # 无新内容
        
        # 使用简单的文本相似度（可以替换为embedding相似度）
        from difflib import SequenceMatcher
        similarity = SequenceMatcher(None, prev_summary, new_summary).ratio()
        novelty = 1.0 - similarity
        
        return max(0.0, min(1.0, novelty))
    
    def _calculate_info_gain(self, topic: str, findings: dict, kg) -> float:
        """计算信息增益"""
        # 基于探索结果的丰富程度
        summary_length = len(findings.get("summary", ""))
        sources_count = len(findings.get("sources", []))
        
        # 归一化
        length_score = min(1.0, summary_length / 1000)  # 1000字符为满分
        sources_score = min(1.0, sources_count / 5)  # 5个来源为满分
        
        return (length_score + sources_score) / 2
    
    def _compute_quality_score(self, novelty: float, info_gain: float) -> float:
        """计算最终质量分"""
        # 权重：新颖度 60%，信息增益 40%
        base_score = novelty * 6 + info_gain * 4
        
        # 映射到 0-10
        return round(base_score, 1)

        self._consecutive_empty = 0
        self._max_empty_before_sleep = 5
        self._sleep_interval = 30  # 秒
        
    def run(self):
        """主循环"""
        logger.info("[SpiderAgent] Started")
        
        while self.running:
            try:
                # 1. 尝试 claim 下一个任务
                item = self._queue.claim_next(agent_id=self.name)
                
                if item:
                    self._consecutive_empty = 0
                    self._process_item(item)
                else:
                    # 队列为空
                    self._consecutive_empty += 1
                    logger.debug(f"[SpiderAgent] No items available (consecutive: {self._consecutive_empty})")
                    
                    if self._consecutive_empty >= self._max_empty_before_sleep:
                        logger.info(f"[SpiderAgent] Queue empty for {self._consecutive_empty} cycles, sleeping...")
                        time.sleep(self._sleep_interval)
                    else:
                        time.sleep(1)  # 短暂等待
                        
            except Exception as e:
                logger.error(f"[SpiderAgent] Error in main loop: {e}", exc_info=True)
                time.sleep(5)  # 出错后等待
        
        logger.info("[SpiderAgent] Stopped")
    
    def _process_item(self, item: QueueItem):
        """处理单个队列项"""
        topic = item.topic
        logger.info(f"[SpiderAgent] Processing: {topic}")
        
        try:
            # 2. 标记开始探索
            if not self._queue.start_exploration(item.id, self.name):
                logger.warning(f"[SpiderAgent] Failed to start exploration for {topic}")
                return
            
            # 3. 执行探索
            result = self._explorer.explore(
                topic=topic,
                depth=item.depth,
            )
            
            # 4. 评估质量
            quality = self._assess_quality(topic, result)
            
            # 5. 完成探索
            exploration_result = ExplorationResult(
                topic=topic,
                summary=result.get("summary", ""),
                sources=result.get("sources", []),
                quality=quality,
                findings=result,
            )
            
            if quality >= 7.0:
                # 高质量完成
                self._queue.complete_exploration(item.id, exploration_result)
                logger.info(f"[SpiderAgent] Completed {topic} with quality {quality}")
                
                # 触发分解
                self._decompose_and_enqueue(item, result)
            else:
                # 低质量，标记完成但不分解
                self._queue.complete_exploration(item.id, exploration_result)
                logger.info(f"[SpiderAgent] Completed {topic} with low quality {quality}, skipping decomposition")
                
        except Exception as e:
            logger.error(f"[SpiderAgent] Error processing {topic}: {e}", exc_info=True)
            
            # 标记失败（会自动处理重试）
            self._queue.fail_exploration(
                item_id=item.id,
                error=str(e),
                retryable=True,
            )
    
    def _assess_quality(self, topic: str, result: dict) -> float:
        """评估探索质量"""
        from core.quality_v2 import QualityV2Assessor
        
        assessor = QualityV2Assessor(self._explorer.llm_client)
        findings = {
            "summary": result.get("summary", ""),
            "sources": result.get("sources", []),
        }

    """
    统一错误处理器
    """
    
    @staticmethod
    def handle_queue_error(error: Exception, context: dict) -> 'ErrorResult':
        """处理队列错误"""
        logger.error(f"[QueueError] {error}, context: {context}")
        
        if isinstance(error, FileNotFoundError):
            # 文件不存在，尝试恢复
            return ErrorResult(
                action="retry",
                delay=1.0,
                max_retries=3,
            )
        
        elif isinstance(error, PermissionError):
            # 权限错误，无法自动恢复
            return ErrorResult(
                action="alert",
                message="Permission denied accessing queue file",
            )
        
        elif isinstance(error, json.JSONDecodeError):
            # JSON 损坏，尝试从备份恢复
            return ErrorResult(
                action="restore_from_backup",
                message="Queue file corrupted",
            )
        
        else:
            # 未知错误，重试
            return ErrorResult(
                action="retry",
                delay=5.0,
                max_retries=3,
            )

## 五、风险管控（修订版）

### 5.1 风险矩阵（修订）

| 风险 | 概率 | 影响 | 缓解措施 | 应急方案 |
|------|------|------|---------|---------|
| 数据迁移失败 | 中 | 高 | 完整备份 + dry-run | 立即回滚到旧系统 |
| 性能下降 >30% | 低 | 中 | 压力测试 + 分段锁 | 启用只读缓存 |
| 状态机死锁 | 低 | 高 | 超时机制 + 分段锁 | 重启 + 状态恢复 |
| 数据不一致 | 中 | 高 | ConsistencyMonitor | 自动修复 + 告警 |
| 兼容性问题 | 中 | 中 | Feature Toggle | 立即切回旧实现 |
| 内存泄漏 | 低 | 中 | StateArchiver | 重启服务 |

### 5.2 详细应急方案

#### 场景 1: 数据迁移失败
```
1. 停止迁移脚本
2. 恢复数据库备份
3. 切回 Feature Toggle: v2_queue_enabled = false
4. 验证旧系统正常
5. 分析问题，修复后重试
```

#### 场景 2: 状态机死锁
```
1. 检测：TimeoutMonitor 发现大量超时
2. 诊断：检查锁状态
3. 恢复：强制释放所有 CLAIMED 状态
4. 重启：优雅重启服务
5. 预防：调整锁粒度或超时时间
```

#### 场景 3: 性能下降
```
1. 监控：MetricsCollector 发现延迟增加
2. 诊断：定位性能瓶颈
3. 缓解：启用只读缓存 / 增加 worker 数量
4. 修复：优化代码
5. 验证：性能回归测试
```

## 六、总结

### 6.1 修订版核心改进

| 方面 | 原方案 | 修订版 |
|------|--------|--------|
| **并发控制** | threading.Lock | 分段锁 + 文件锁 |
| **持久化** | 接口未实现 | 完整 JSONRepository |
| **超时处理** | 未实现 | TimeoutMonitor |
| **数据迁移** | 策略不具体 | 完整迁移脚本 + dry-run |
| **内存管理** | 未考虑 | StateArchiver |
| **错误处理** | 不完整 | 统一错误处理 |
| **测试** | 缺失 | 完整测试策略 |
| **发布** | 一次性 | 灰度发布 |

### 6.2 关键保障措施

1. **双写期**: 新旧系统并行运行，确保数据安全
2. **Feature Toggle**: 随时可切回旧系统
3. **自动备份**: 每次写操作前自动备份
4. **一致性检查**: 实时监控数据一致性
5. **灰度发布**: 逐步放量，降低风险
